# Z-Paste - 需求文档

## 功能需求

### FR-1: 剪贴板监听

| ID | 描述 |
|----|------|
| FR-1.1 | 后台监听系统剪贴板变化 |
| FR-1.2 | 支持文本、图片、文件、富文本类型 |
| FR-1.3 | 可配置排除应用列表 |
| FR-1.4 | 去重逻辑（相同内容不重复记录） |

### FR-2: 历史管理

| ID | 描述 |
|----|------|
| FR-2.1 | 保留最近 1000 条记录 |
| FR-2.2 | 自动清理超出限制的历史 |
| FR-2.3 | 支持收藏/置顶功能 |
| FR-2.4 | 收藏项不受数量限制清理 |

### FR-3: 界面交互

| ID | 描述 |
|----|------|
| FR-3.1 | 底部弹出式窗口动画 |
| FR-3.2 | 横向滚动卡片布局 |
| FR-3.3 | 卡片显示：类型、时间、来源图标、预览、字符数 |
| FR-3.4 | 选中高亮效果 |
| FR-3.5 | 点击卡片复制/粘贴 |

### FR-4: 搜索功能

| ID | 描述 |
|----|------|
| FR-4.1 | 文本内容搜索 |
| FR-4.2 | 实时过滤结果 |
| FR-4.3 | 搜索高亮 |

### FR-5: 快捷键

| ID | 描述 |
|----|------|
| FR-5.1 | 默认 Option + ` 唤起/隐藏 |
| FR-5.2 | 设置中可自定义快捷键 |
| FR-5.3 | 全局快捷键监听 |

### FR-6: 设置

| ID | 描述 |
|----|------|
| FR-6.1 | 快捷键配置 |
| FR-6.2 | 历史数量限制配置 |
| FR-6.3 | 应用排除列表管理 |
| FR-6.4 | 外观设置（主题、卡片大小） |

## 非功能需求

### NFR-1: 性能

- 唤起响应时间 < 100ms
- 内存占用 < 100MB
- 启动时间 < 2s

### NFR-2: 隐私

- 所有数据本地存储
- 无网络请求
- 敏感应用内容自动排除

### NFR-3: 兼容性

- macOS 13.0+
- 支持 Intel 和 Apple Silicon
