# Z-Paste - macOS 剪贴板管理器

## 项目愿景

打造一款轻量、优雅、隐私优先的 macOS 原生剪贴板管理工具，灵感来源于 Paste app。

## 核心特性

### 功能需求

| 特性 | 描述 |
|------|------|
| 📋 历史记录 | 保留 500-1000 条剪贴板历史 |
| 🎨 卡片预览 | 横向滚动卡片式布局，内联内容预览 |
| ⚡ 快速唤起 | Option + ` 快捷键从底部弹出 |
| 🔒 隐私保护 | 可排除特定应用（密码管理器等） |
| ⭐ 收藏功能 | 标记常用内容为收藏/置顶 |
| 🔍 基础搜索 | 按文本内容搜索历史记录 |
| 📎 多类型支持 | 文本、图片、文件、富文本 |
| 🏷️ 来源标识 | 显示复制来源应用图标 |

### 技术规格

- **平台：** macOS 13.0+
- **技术栈：** Swift 5.9+ / SwiftUI
- **架构：** MVVM + 依赖注入
- **存储：** SQLite (本地) + UserDefaults (配置)
- **快捷键：** 全局可配置快捷键
