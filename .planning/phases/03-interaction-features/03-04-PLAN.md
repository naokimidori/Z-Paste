    1. 保留 `setupClickOutsideHandling()` 的 global/local monitor 结构，但在两个监视器的最前面加入 `if isContextMenuPresented { return }` 或等价分支；不要整体移除 click-outside 能力。
    2. 对本地监视器的 inside/outside 判断继续基于 `panel.frame.contains(locationOnScreen)`；菜单状态只负责“菜单期间不隐藏”，不改 panel 几何判断规则。
