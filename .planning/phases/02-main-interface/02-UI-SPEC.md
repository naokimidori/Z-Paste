## Color

### Color Distribution

| Role | Value | Usage |
|------|-------|-------|

### Accent Reserved For

- Selected card border stroke (2px)
- Keyboard focus indicator
- Active state indicators

**NOT used for:** card backgrounds, text, icons (except active states)

## Interaction Contract

### Keyboard Navigation

| Key | Action | Notes |
|-----|--------|-------|
| Option + ` | Toggle window show/hide | Global hotkey |
| Left Arrow | Select previous card | Wrap to last if at first |
| Right Arrow | Select next card | Wrap to first if at last |
| Enter | Select and copy current card | Auto-close after selection |
| Escape | Close window | No selection |

### Pre-populated From CONTEXT.md

| Parameter | Value | Source |
|-----------|-------|--------|
| Keyboard navigation | Left/right arrow keys, Enter confirm | CONTEXT.md (交互方式) |
| Default selection | Newest card (leftmost) | CONTEXT.md (交互方式) |
| Sort order | Reverse chronological, newest first | CONTEXT.md (排序规则) |
| Close triggers | Hotkey toggle, click outside, Escape, auto-close after select | CONTEXT.md (动画效果) |

### Window Behavior

| Trigger | Behavior |
|---------|----------|
| Global hotkey pressed | Toggle: show with slide animation OR hide with slide animation |
| Window shown | Activate app, focus first card, load recent history |
| Window hidden | Deactivate, clear selection state |
| Click outside window | Hide window with slide animation |
| Card selected (Enter) | Copy content to clipboard, hide window |

### Animation Specifications

| Animation | Duration | Easing | Notes |
|-----------|----------|--------|-------|
| Window slide in | 0.2s | easeInEaseOut | From bottom of screen |
| Window slide out | 0.2s | easeInEaseOut | To bottom of screen |
| Card selection | 0.15s | easeOut | Border color transition |
| Card hover | 0.1s | easeOut | Opacity change |

### Pre-populated From CONTEXT.md

| Parameter | Value | Source |
|-----------|-------|--------|
| Popup animation | Slide from bottom | CONTEXT.md (动画效果) |
| Dismiss animation | Slide out to bottom | CONTEXT.md (动画效果) |
| Animation duration | 0.2 seconds | CONTEXT.md (动画效果) |

## Visual Effects

### Glass Morphism (Liquid Glass)

| Element | Effect | SwiftUI Implementation |
|---------|--------|------------------------|

### State Visual Feedback

| State | Visual Change |
|-------|---------------|

| Hovered | Slight opacity reduction (0.9) |
